## ◈ Featured Projects

<details>
<summary><strong>🤖 Clone Pose — Real-Time Human Pose Replication</strong></summary>

<br>

> Real-time AI system that detects and replicates human body pose using computer vision and skeletal landmark mapping.

| Attribute | Detail |
|---|---|
| **Stack** | Python · OpenCV · MediaPipe · NumPy |
| **Scale** | Real-time processing at 30+ FPS on standard hardware |
| **Performance** | Sub-50ms inference latency with 33-point skeletal tracking |
| **Security** | Local inference, no cloud dependency, privacy-preserving |
| **Impact** | State-Level Project Competition award winner; demonstrated at DipNova 2026 |
| **Repository** | [github.com/SairajAgawane](https://github.com/SairajAgawane) |

This project leverages MediaPipe's pose estimation pipeline to capture real-time body landmarks from a webcam feed and replicate the pose on a virtual avatar. The system achieves smooth frame rates through optimized OpenCV rendering loops and landmark smoothing algorithms, making it suitable for fitness coaching, animation mirroring, and accessibility applications.

</details>
